Challenge

The seed to Code Excursion came the summer of 2017 when we (Louise and Corina) had been trying to learn coding for a long time without much success. We both felt like we were missing a safe and inspiring space to learn. To fix this problem we created Code Excursion!

Rule Book


Open data is data that can be freely used, re-used and redistributed by anyone. Several agencies contributes to Hack for Sweden with open data. To participate in Hack for Sweden you must use data from at least one of the contributing authorities, but feel free to combine government data with other data sources.
